# Privacy Policy for Snappix

Last updated: May 2026

Snappix uses Google OAuth to allow users to sign in to their Google account and access read‑only Google Classroom information.

## Information We Access
Snappix requests the following Google scopes:
- `email`
- `profile`
- `classroom.courses.readonly`
- `classroom.rosters.readonly`

This allows Snappix to:
- Identify the signed‑in user
- Display the user’s name 
- Read course lists and rosters from Google Classroom
- Add student names in the selected courses to a token that used for picking names
- Note: Student names are encrypted and saved locally so Google Classroom is not required after Setup

## Information We Do NOT Collect
Snappix does **NOT**:
- Store ANY data on external servers
- Transmit data to third parties
- Collect analytics
- Sell or share any personal information
- Save unencrypted data to any permanent storage

All data is processed **locally on the user’s device**. Locally stored student information is encrypted using AES256.
